PokerSim

Advanced Poker Simulation Framework with ML Integration

PokerSim is a comprehensive Python API for poker simulation, designed for machine learning research and game theory exploration. With advanced opponent modeling, ML framework integration, and performance optimizations.

Key Features

  • Advanced opponent agents with sophisticated playing styles
  • Spin and Go tournament mode
  • Deep Counterfactual Regret Minimization (DCFR) implementation
  • Proximal Policy Optimization (PPO) for reinforcement learning
  • Performance optimization with Numba
  • PyTorch and TensorFlow integration
