About PokerSim

Project Overview

PokerSim is a comprehensive Python API designed for advanced poker simulation and reinforcement learning research.

This framework provides a robust platform for simulating poker games with various agent types, from basic random players to sophisticated AI opponents. It's designed to be modular, extensible, and optimized for reinforcement learning research in the poker domain.

Core Components

Game Engine

A fast, efficient poker simulation engine supporting multiple game variants including Texas Hold'em and Spin and Go.

  • Full game state management
  • Card and deck handling
  • Hand evaluation and showdown logic
  • Betting rounds and action validation
Agent System

A flexible agent architecture supporting various playing styles and strategies.

  • Basic agents: Random, Call, Fold
  • Rule-based agents with hand strength evaluation
  • Advanced opponents with sophisticated modeling
  • ML-based agents using reinforcement learning
ML Integration

Seamless integration with popular machine learning frameworks.

  • PyTorch integration for deep learning
  • TensorFlow support for neural networks
  • Reinforcement learning algorithms (PPO, DCFR)
  • Feature extraction and representation
Performance Optimization

High-performance computing features for efficient simulations.

  • Numba acceleration for critical operations
  • Parallelized simulations for training
  • Memory-efficient data structures
  • Optimized hand evaluation algorithms
Technical Specifications

Language: Python 3.8+

  • PyTorch
